# 硬件资源与接口映射

## 1. 硬件资源

### 1.1 主控

- 芯片：ESP32-P4（Xtensa LX7 双核 + HP/LP RAM，无 RF）
- 开发板：JC-ESP32P4-M3-DEV
- 主要片上外设：MIPI CSI、MIPI DSI、I2C×3、I2S×3、SPI、UART、SDMMC、Ethernet MAC、USB

### 1.2 显示器

- 型号：RPI_7INCH_DSI_DISPLAY_V1（Raspberry Pi 7" Touch Display V1）
- 接口：MIPI DSI（经 TC358762 桥接为 DPI 面板）
- 分辨率：800 × 480
- 背光/电源管理：板载 ATtiny88 MCU（经 I2C 控制，非直接 GPIO PWM）

### 1.3 摄像头

- 型号：OV5647
- 接口：MIPI CSI
- SCCB 控制总线：与系统 I2C0 共用

### 1.4 人体红外感应模块

- 型号：HC-SR501
- 接口：GPIO 输入 + 外部中断

### 1.5 存储

- 型号：TF-card（microSD）
- 接口：SDMMC 4-bit SDIO 模式
- 挂载点：`/sdcard`（人脸识别模型、用户库、日志存储）

### 1.6 音频模块

- Codec：ES8311
- 接口：I2S（数据流）+ I2C（控制寄存器）
- 功放：板载 PA，独立使能引脚

### 1.7 触摸屏

- 控制器：FT5406（Raspberry Pi 7" 屏集成）
- 接口：I2C
- 分辨率：800 × 480

### 1.8 网络（有线以太网）

- PHY：IP101（板载 RJ45）
- 接口：RMII（ESP32-P4 内置 MAC + 外部 PHY）
- 用途：MQTT 通信、上位机对接

### 1.9 其他

| 模块 | 型号/说明 | 接口 | 控制引脚 | 有效电平 |
| ---- | ---- | ---- | ---- | ---- |
| 继电器 | 门锁驱动 | GPIO 输出 | GPIO2 | 可配（`active_level`） |

> 继电器吸合时输出 `active_level`，释放时输出其反相电平；具体高低有效由上层配置决定。

### 2.8 以太网接口映射表

| PHY 地址 | 1 | `NETWORK_ETH_PHY_ADDR` | ✅ |
| 接口 | RMII | `CONFIG_ETH_PHY_INTERFACE_RMII=y` | ✅ |
| MDC | GPIO31 | `NETWORK_ETH_MDC_PIN`（SMI 时钟） | ✅ |
| MDIO | GPIO52 | `NETWORK_ETH_MDIO_PIN`（SMI 数据） | ✅ |
| PHY RST | GPIO51 | `NETWORK_ETH_PHY_RESET_PIN` | ✅ |

## 5. 待确认与风险事项

| 项 | 说明 | 建议 |
